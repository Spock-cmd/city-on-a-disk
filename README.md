# City on a Disk Drive
This program allows you to generate a city in your empty disk drive.

Before running the program, make sure the following conditions are met:
- You have an empty storage device (e.g. Disk partition or a USB drive)
- The storage device is empty

## Steps to run the program:
1. Extract the ```City-on-a-Disk``` folder
2. Navigate to the extracted folder
3. Run the program (if it is asking for confirmation, click YES.)
4. Wait for the program to load
5. Type the drive letter of the empty disk where you want to generate the city
6. Enjoy your new city!

## Troubleshooting
If there is no YES button if it asks for confirmation, you need to have administrator\
priveliges.

To enable admin privelidges, follow the steps
1. Open the start menu and search for ```control panel```
2. In the Control Panel, click on ```User Accounts```
3. find and click ```Change Account Type```
4. Find your Account
5. Click on ```Change the Account type```
6. Click on ```Administrator```, then click ```Change Account Type```
7. Restart the PC if needed
